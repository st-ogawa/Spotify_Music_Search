# Music Search　

SpotifyAPIを利用した楽曲検索サイトです。
お気に入りの楽曲を見つけたらお気に入り登録ができて、楽曲の詳細ページに飛ぶことができます。
　

## Dependency

```
laravel: 6.*
laravel/passport: 9.*
laravel/ui: 1.0
axios: 0.19
vue: 2.*
vuex : 3.6.2
vue-router :3.5.1
vuejs-paginate : 2.1.0

```


## Setup

```

$ git clone https://github.com/B-FORME/Spotify_Ogawa.git
$ npm install
$ npm run serve

```
